Posted on 2020-04-17 | In 算法 |

散列技术是在记录的存储位置和它的关键字之间建立一个确定的对应关系 f，使得每个关键字 key 对应一个存储位置 f(key)。其中 f 称为散列函数，又称哈希函数。按照这个思想，采用散列技术将记录存储在一块连续的存储空间中，这块连续存储空间称为散列表或哈希表。那么关键字对应的记录存储位置为散列地址。

哈希表

一、基本定义

哈希表（也叫 散列表），是根据关键码值而直接进行访问的数据结构。它通过把关键码值映射到表中一个位置来访问记录，以加快查找的速度，这个映射函数叫作 散列函数，存放记录的数组叫作 散列表。

二、实际应用

使用哈希表管理水浒英雄。其中每个人的id（排名）唯一，即为上图中的keyVal关键码值，除此之外还有 姓名、诨名 作为其他码值。管理操作包括但不限于：顺序添加、删除、查看、更改、遍历显示……

实现主类

哈希表类（通过调用链表的管理操作完成哈希表的增删改查）

英雄结点类

链表类（链表的增删改查）

值得注意的是，该哈希表中的链表的头指针直接指向第一节点，初始头节点为空
