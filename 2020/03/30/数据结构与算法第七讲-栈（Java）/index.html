数据结构与算法第七讲 栈（Java）

Posted on 2020-03-30 | In 算法 |
| Words count in article: 1.6k | Reading time ≈ 7

“类似很多软件，比如Word、Photoshop等，都有撤销（undo）操作，也是用栈这种思想方式来实现的。”

栈

一、基本定义

栈是限定仅在表尾进行插入和删除操作的线性表。允许插入和删除的一端称为 栈顶（top），另一端称为 栈底（bottom），不含有任何数据元素的栈称为 空栈。栈先进而后出。

栈的插入操作，叫作 进栈（push）；栈的删除操作，叫作 出栈（pop）

二、应用场景
  1. 子程序的调用：在跳往子程序前，会先将下一个指令的地址存到堆栈中，直到子程序执行完后再将地址取出，以回到原来的程序中
  2. 处理递归调用：同前者类似，只是除了存储下一个指令的地址外，也将参数、区域变量等数据存入堆栈中
  3. 表达式的转换【中缀表达式转后缀表达式】与计算机底层表达式求值
  4. 二叉树的遍历
  5. 图形的深度优先搜索法
三、数组实现栈

  1. 使用数组来模拟栈

  2. 定义一个top来表示栈顶，初始化为-1

  3. 入栈的操作，当有数据加入到栈时，top++; stack[top]=data;

  4. 出栈的操作，int value =stack[top]; top--; return value;

  5. 遍历的操作，栈在进行遍历时，从栈顶开始向下遍历。

四、单链表实现栈

单链表的实现过程并不复杂，但是要注意当栈结点插入时，应该使用“头插法”（越晚插入的栈结点离头结点越近）

五、计算器

划分两个栈，一个专门存放数值，为数栈；另一个专门存放运算符，为符号栈

  1. 首先通过一个index索引值，来遍历表达式
  2. 如果发现当前为数值，直接入数栈（注意当数值为多位数时，需要判断当前数值之后几位是数值还是运算符；还需判断当前数值是否已是最后一位，否则会引起数值出界）
  3. 如果发现当前为运算符，分如下情况
    • 如果当前符号栈为空，直接入栈
    • 如果当前符号栈有操作符，进行如下比较
      • 当前运算符的优先级小于等于符号栈栈顶的运算符，从数栈中pop两个数，符号栈pop一个运算符，进行运算，将所得结果入数栈，然后当前运算符如符号栈
      • 如果当前运算符优先级大于符号栈栈顶的运算符，直接入符号栈
  4. 当表达式扫描完毕，就顺序从数栈和符号栈中pop出相应的数值和符号，并运行计算
  5. 当在数栈中只有一个数值，符号栈空，该数值就是表达式的结果
