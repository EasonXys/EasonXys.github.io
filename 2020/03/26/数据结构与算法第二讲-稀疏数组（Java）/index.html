稀疏数组

一、应用场景

当数组中大部分元素为0，或者为同一值的数组时，可以使用稀疏数组来表示

二、处理方法

1）记录原二维数组大小（几行几列），有多少个不同的值

2）把具有不同值的元素行、列、值记录在 小规模数组 （即稀疏数组）中

​ 第一行中，由左及右分别记录总行数、总列数、不同值的个数

​ 第二行中，记录非零值的行、列索引及值

三、相互转换思路步骤

二维数组转换为稀疏数组

  1. 遍历原始二维数组 chessArray1，得到有效数据个数sum；
  2. 根据sum创建稀疏数组 int spareseArray[][]=new int[sum+1][3];
  3. 将 chessArray1 有效数据存入 spareseArray 中。

稀疏数组还原为二维数组

  1. 先读取 spareseArray 第一行信息创建二维数组 chessArray2；

  2. 读取 spareseArray 后几行数据，并赋值给 chessArray2。
