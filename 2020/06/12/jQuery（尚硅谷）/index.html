jQuery（尚硅谷）

Posted on 2020-06-12 | In 语言 |

什么是 jQuery ？

j 就是JavaScript，Query 查询，意思就是查询js，把js中的DOM操作（CRUD、Ajax）做了封装，可以快速查询使用里面的功能。

jQuery是一个JavaScript函数库，是一个轻量级的”write less, do more“的JavaScript库。

jQuery库包含以下功能：

  • HTML 元素选取

  • HTML 元素操作

  • CSS 操作

  • HTML 事件函数

  • JavaScript 特效和动画

  • HTML DOM 遍历和修改

  • AJAX

  • Utilities

提示： 除此之外，jQuery还提供了大量的插件。

jQuery

一、基础认识

jQuery优点

  • 轻量级。核心文件才几十kb，不会影响页面加载速度
  • 跨浏览器兼容。基本兼容现在主流浏览器
  • ⭐链式调用、读写合一（读写数据共用一个方法）、隐式遍历（一次操作多个元素）
  • 对事件、样式、动画支持，大大简化DOM操作（CRUD）
  • 支持插件扩展开发。有着丰富的第三方插件，比如：树形菜单、日期控件、轮播图等
  • 免费、开源

jQuery引入

  • 本地引入与CDN远程引入
  • 测试版与生产版(压缩版)

jQuery的入口函数

等着DOM结构渲染完毕即可执行内部代码，不必等到所有外部资源加载完成，且可以有多个监听回调，jQuery帮我们完成了封装。相当于原生js中的DOMContentLoaded。与原生js中的load事件（等页面文档、外部的js文件、css文件、图片加载完毕之后才执行内部代码，且只能有一个监听回调）不同

window.onload 和 $(document).ready()的区别

  • $(document).ready()

    采用的是DOM2事件绑定，监听的是DOMContentLoaded这个事件，所以只要DOM结构加载完成就会被触发执行，而且同一个页面中可以使用多次（绑定不同的方法，因为基于DOM2事件池绑定机制完成的）

  • window.onload

    必须等待所有资源都加载完成才会被触发执行，采用DOM0事件绑定，同一个页面只能绑定一次（一个方法），想绑定多个也需要改为window.addEventListener('load', function () {})DOM2绑定方式

$是jQuery的别称，在代码中也可以直接使用jQuery。

二、jQuery函数和jQuery对象

jQuery核心函数

jQuery核心函数是jQuery()或$()的值，jQuery类库定义了这个全局函数供使用者调用该函数，可以用来创建jQuery对象。它既可以作为一般函数调用，且传递的参数类型不同、格式不同，功能也就完全不同，也可以作为对象调用其定义好的方法，此时的$就是一个工具对象，类似于Math。

当函数用：$(xxx)

当工具对象用：$.xxx()

jQuery核心对象

jQuery()是工厂函数，而非构造函数，执行它返回的一个新创建的对象，但并没有使用new关键字一起使用。jQuery对象内部包含的是DOM元素对象的伪数组，它定义了很多方法，可以用来操作它们表示的这组元素。

  • jQuery函数的使用
    • 作为一般函数调用：$(param)
      • param为匿名函数：当DOM加载完成之后，执行此回调函数
      • param为选择器字符串：查找所有匹配的DOM元素，并将它们封装成jQuery对象返回
      • param为DOM元素：将DOM元素封装成jQuery对象返回
      • param为HTML标签字符串：创建标签对象并封装成jQuery对象返回
    • 作为工具对象使用：$.xxx()
      • $.each()：隐式遍历数组
      • $.trim()：去除两端空格
  • jQuery对象的使用：jQuery对象.属性/方法
    • 基本行为
      • size()/ length：取出jQuery对象内部包含的DOM元素的个数
      • [index]/ get(index)：取出jQuery对象内部对应下标的DOM元素
      • each(function(index,domEle){})：遍历所有DOM元素
      • index()：得到当前DOM元素在所有兄弟元素中的下标
    • 属性：操作内部标签的属性或值
    • CSS：操作标签的样式
    • 文档：对标签进行增删改操作
    • 筛选：根据指定的规则过滤内部的标签
    • 事件：处理事件监听相关
    • 效果：实现一些动画效果

伪数组：是一个具有 length属性 和 数值下标属性，但没有数组特有方法的对象

三、jQuery核心函数的使用

选择器

选择器本身只是一个有特定语法规则的字符串，没有实质意义。它的基本语法规则使用的就是CSS的选择器语法，并对其进行了扩展。只有调用$(selector)，才能真正起到作用：根据选择器规则在整个文档中查找所有匹配的标签的数组，并封装成jQuery对象返回。

选择器分类：

  • 基本选择器：最基本常用的CSS选择器
    • #id, element, .class, *, selector1,selector2 并集选择, selector1selector2 交集选择 ……
  • 层次选择器：查找后代元素、子元素、兄弟元素的选择器
    • ancestor descendant
    • parent > child
    • prev + next
    • prev ~ siblings 匹配prev元素之后的所有siblings兄弟元素
  • 过滤选择器：在原有选择器匹配的元素中经过进一步筛选得到的选择器，多个过滤选择器不是同时执行的，而是依次执行
    • :first
    • :last
    • :not(selector)
    • :odd
    • :even
    • :gt()：匹配所有大于index的下标元素
    • :lt()：匹配所有小于index的下标元素
    • :contains("content")：匹配所有包含内容为content的元素
    • :hidden()
    • [属性名]
    • [属性名=属性值]
  • 表单选择器
    • 表单
      • :input
      • :password
    • 表单属性
      • :disabled
      • :checked
$工具对象的方法
  • $.each()
  • $.trim()
  • $.type(obj)
  • $.isArray(obj)
  • $.isFunction(obj)
  • $.parseJSON(json)：解析json字符串转换为js对象/ 数组

四、jQuery核心对象的使用

属性（操作标签的属性、标签体文本）
  • attr(属性名) / attr(属性名, 属性值)：读/ 写非布尔值的标签属性
  • prop(属性名) / prop(属性名, 属性值)： 读/ 写布尔值的标签属性
  • removeAttr(属性名)/removeProp(属性名)： 删除属性
  • addClass(classValue)：添加class
  • removeClass(classValue)： 移除指定class
  • val() / val(value)： 读/ 写标签的value
  • html() / html(htmlString)： 读/ 写标签体文本
CSS模块
  • style样式
    • css(styleName): 根据样式名得到对应的值
    • css(styleName, value): 设置一个样式
    • css({多个样式对}): 设置多个样式
  • 位置坐标
    • offset(): 读/写当前元素坐标(原点是页面左上角)
    • position(): 读当前元素坐标(原点是父元素左上角)
    • scrollTop()/scrollLeft(): 读/写元素/页面的滚动条坐标
  • 尺寸
    • width()/height(): width/height
    • innerWidth()/innerHeight(): width + padding
    • outerWidth(false|true)/outerHeight(false|true): width + padding + border，如果是true，再加上margin
筛选模块
  • 过滤：在jQuery对象内部的元素中找出部分匹配的元素, 并封装成新的jQuery对象返回
    • first()
    • last()
    • eq(index)
    • filter(selector): 对当前元素提要求
    • not(selector): 对当前元素提要求, 并取反
    • has(selector): 对后代元素提要求
  • 查找：查找jQuery对象内部的元素的子孙/兄弟/父母元素, 并封装成新的jQuery对象返回
    • children(selector): 子元素
    • find(selector): 后代元素
    • preAll(selector): 前的所有兄弟
    • siblings(selector): 所有兄弟
    • parent(): 父元素

文档处理
  • 增加
    • append(要追加到目标中的内容) / appendTo(被追加的内容): 向当前匹配的所有元素内部的最后插入指定内容
    • preppend(要追加到目标中的内容) / preppendTo(被追加的内容): 向当前匹配的所有元素内部的最前面插入指定内容
    • before(要追加到目标中的内容): 将指定内容插入到当前所有匹配元素的前面
    • after(要追加到目标中的内容): 将指定内容插入到当前所有匹配元素的后面
  • 删除
    • remove(): 将自己及内部的孩子都删除
    • empty(): 掏空(自己还在)
  • 更新
    • replaceWith(HTML标签字符串)
    • replaceAll(选择器字符串)

注意：append/prepend 是在选择元素内部嵌入。after/before 是在元素外面追加。

事件模块

  • 绑定事件，常用事件eventName: click, mouseenter/mouseleave mouseover/mouseout focus/blur

    • eventName(function(){}) 编码简单，但不是所有的事件都有对应的方法
    • on('eventName', function(){}) 编码较麻烦，但是通用
    • hover(function(){}, function(){})：事件切换，第一个function是鼠标移入对应的事件，第二个是鼠标移出对应的事件

  • 解绑事件

    • off('eventName')

  • 事件坐标

    • event.offsetX: 原点是当前事件元素左上角
    • event.clientX: 原点是窗口左上角
    • event.pageX: 原点是当前页面左上角

  • 事件相关

    • 停止事件冒泡: event.stopPropagation()
    • 阻止事件的默认行为:event.preventDefault()

  • ⭐事件委托

    • 理解: 将子元素的事件委托给父辈元素处理
      • 事件监听绑定在父元素上, 但事件发生在子元素上
      • 当操作任何一个子元素时，事件会冒泡到父元素
      • 父元素不会直接处理事件，而是根据:event.target得到发生事件的子元素，并通过这个子元素调用事件回调函数
    • 好处
      • 新增的元素也可以自动获取事件监听处理
      • 减少事件监听的数量(n==>1)
    • 编码
      • delegate(selector, 'eventName', function(event){})
      • undelegate('eventName')：移除事件委托

注意：on('eventName',selector,event.data,function(event){}) 在jQuery 1.7中，.on()方法 提供绑定事件处理程序所需的所有功能。帮助从旧的jQuery事件方法转换，see .bind(), .delegate(), 和 .live()。 要删除的.on()绑定的事件，请参阅.off()。

事件模块面试题

  1. 区别mouseover和mouseenter ？

    mouseover：在移入子元素时也会触发，对应mouseout

    mouseenter：只在移入当前元素时才会触发，对应mouseleave。hover()用的就是mouseenter()，mouseleave()

动画效果

在一定的时间内, 不断改变元素样式

  • slideDown()/slideUp()/slideToggle()：滑入滑出
  • fadeOut()/fadeIn()/fadeToggle()：淡入淡出，即不断改变元素的透明度来实现
  • show()/hide()/toggle()：显示隐藏
  • animate(对象.{结束时的样式}, time, function(){})
  • stop()

五、jQuery多库共存

如果有两个库都有$，就存在冲突。jQuery库可以释放$的使用权jQuery.noConflict();，让另一个库可以正常使用，此时jQuery库就只能使用jQUery了
