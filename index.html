什么是 jQuery ？

j 就是JavaScript，Query 查询，意思就是查询js，把js中的DOM操作（CRUD、Ajax）做了封装，可以快速查询使用里面的功能。

jQuery是一个JavaScript函数库，是一个轻量级的”write less, do more“的JavaScript库。

jQuery库包含以下功能：

  • HTML 元素选取

  • HTML 元素操作

  • CSS 操作

  • HTML 事件函数

  • JavaScript 特效和动画

  • HTML DOM 遍历和修改

  • AJAX

  • Utilities

提示： 除此之外，jQuery还提供了大量的插件。

JavaScript的基本介绍

ECMAScript是JavaScript（一门基于对象的、运行在客户端的脚本语言）标准，一般情况下认为是一个意思。一个完整的JavaScript实现应该由以下三个部分构成：

  • ECMAScript：由ECMA-262定义，提供核心语言功能
  • DOM：将页面映射为一个多层节点结构，并提供访问和操作网页内容的方法和接口
  • BOM：提供与浏览器交互的方法和接口

JavaScript的特点

  • 解释性语言（无需编译，写完直接由上向下运行）

  • 类似于C和Java的语法结构

  • 动态语言（动态语言是在运行时确定数据类型的语言。变量使用之前不需要类型声明，通常变量的类型是被赋值的那个值的类型。 ）

  • 基于原型的面向对象

script的放置位置

虽然浏览器是多线程的，但是JS是单线程的，如果在<head>标签中添加所有JavaScript文件，就意味着必须等到全部JavaScript代码都被下载，解析和执行完成以后，才能呈现页面的内容（浏览器在遇到<body>标签时才开始呈现内容，而在此之间，浏览器窗口将是一片空白。为了不引起浏览器阻塞，建议将<script>标签放置在页面尾部）

Posted on 2020-04-17 | In 算法
| Words count in article: 1.8k | Reading time ≈ 7

顺序查找

二分查找：在有序表中，取中间记录作为比较对象，如果给定值与中间记录的关键字相等，则查找成功；若给定值小于中间记录的关键字，则在中间记录的左半区继续查找（进行递归）；若给定值大于中间记录的关键字，则在中间记录的右半区进行查找（进行递归）。不断重复上述过程，直至查找成功，或所有查找区域无记录，查找失败为止。

插值查找：插值查找算法类似于二分查找，不同的是插值查找每次从自适应mid处开始查找

斐波那契（黄金分割）查找：斐波那契查找原理与前两种相似，仅仅改变了中间结点（mid）的位置，mid不再是二分或插值得到，而是位于黄金分割点附近，即mid=left+F(k-1)-1（F代表斐波那契数列）

数据结构与算法第十一讲 排序二（快速、归并、基数）（Java）

Posted on 2020-04-13 | In 算法
| Words count in article: 2.3k | Reading time ≈ 9

快速排序：通过一趟排序将待排序的数据分割成两个部分，其中一部分的所有数据比另一部分的所有数据都要小，然后再按此方法对这两部分的数据分别进行快速排序，整个排序过程可以递归进行，以此达到整个数据变成有序序列

归并排序：假设初始序列含有n个数据，则可以看成n个有序的子序列，每个子序列的长度为1，然后两两归并，得到 [n/2] ([num]表示不小于num的最小整数)个长度为2或1的有序子序列；再两两归并……，如此重复，直至得到一个长度为n的有序序列为止

基数排序：将所有待比较的数值统一为同样的数位长度，数位较短的数前面补零。然后从低位开始，依次进行一次排序。这样从最低位排序一直到最高位排序完成后，数列就变成一个有序序列
